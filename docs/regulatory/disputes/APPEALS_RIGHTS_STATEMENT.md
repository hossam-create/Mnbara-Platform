## 1. Rights Declaration

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   EVERY PARTY HAS THE RIGHT TO APPEAL.                      │
│                                                             │
│   If you disagree with a dispute decision:                  │
│   • You may appeal                                          │
│   • Your appeal will be reviewed by a different human       │

│                                                             │

│                                                             │
```

## 2. Appeal Rights

### 2.1 Universal Right to Appeal

| Right | Statement |
| :--- | :--- |
| **Right exists** | Every party to a dispute may appeal |
| **No conditions** | Appeal right is not conditional on amount or type |
| **No cost** | No fee to file an appeal |
| **No retaliation** | Appealing does not affect standing |

### 2.2 What May Be Appealed

| Appealable | Not Appealable |
| :--- | :--- |
| Dispute outcome | Procedural clarifications |
| Compensation amount | Appeal decisions (after exhausted) |
| Evidence evaluation | Timing of earlier decisions |
| Policy application | Third-party decisions |

### 2.3 Grounds for Appeal

| Ground | Description |
| :--- | :--- |
| **Factual error** | Decision based on incorrect facts |
| **Evidence overlooked** | Relevant evidence not considered |
| **Policy misapplication** | Wrong policy applied |
| **Procedural error** | Process not followed |
| **New evidence** | Material evidence now available |
| **Disproportionate outcome** | Penalty exceeds offense |

## 3. Appeal Process

### 3.1 How to Appeal

```
APPEAL SUBMISSION PROCESS

Step 1: LOCATE
└─ Find the dispute decision in your account
└─ Click "Appeal This Decision"

Step 2: SELECT GROUNDS
└─ Choose your reason for appealing
└─ You may select multiple grounds

Step 3: EXPLAIN
└─ Describe why you believe the decision was wrong
└─ Be specific about what should change

Step 4: EVIDENCE (optional)
└─ Submit new evidence if applicable
└─ Explain why it wasn't available before

Step 5: SUBMIT
└─ Review your appeal
└─ Submit before deadline
└─ Receive confirmation
```

### 3.2 Appeal Deadlines

| Decision Type | Appeal Window |
| :--- | :--- |
| Standard dispute | 30 calendar days |
| Elevated dispute | 30 calendar days |
| High-value dispute | 60 calendar days |
| Critical dispute | 90 calendar days |

### 3.3 Deadline Extensions

| Circumstance | Extension |
| :--- | :--- |
| Documented medical issue | Up to 30 days |
| Technical platform issue | Time lost restored |
| Natural disaster | Case-by-case |
| Late notification | Deadline from notification |

## 4. Appeal Review

### 4.1 Independent Reviewer

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   YOUR APPEAL IS REVIEWED BY A DIFFERENT HUMAN.             │
│                                                             │
│   The appeal reviewer:                                      │
│   • Was NOT involved in the original decision               │
│   • Has equal or higher authority                           │
│   • Reviews the case fresh                                  │
│   • Is NOT influenced by the original reviewer              │
│                                                             │
│   AI DOES NOT DECIDE APPEALS.                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 Review Scope

The appeal reviewer will:

| Action | Description |
| :--- | :--- |
| Review original evidence | All evidence from initial dispute |
| Review new evidence | Any new submissions with appeal |
| Review appeal statement | Appellant's arguments |
| Review original decision | Prior reasoning |
| Apply fresh judgment | Independent determination |

### 4.3 What Reviewer Cannot Do

| Prohibition | Reason |
| :--- | :--- |
| Simply defer to original | Must apply independent judgment |
| Increase penalty | No adverse action for appealing |
| Consider that party appealed | Not a negative factor |
| Consult original reviewer | Independence required |

## 5. Appeal Outcomes

### 5.1 Possible Decisions

| Outcome | Meaning |
| :--- | :--- |
| **Affirmed** | Original decision stands |
| **Modified** | Outcome changed (not necessarily in appellant's favor) |
| **Reversed** | Original decision overturned |
| **Remanded** | Sent back for reconsideration |

### 5.2 Outcome Communication

Every appeal decision includes:

| Element | Content |
| :--- | :--- |
| Decision | Clear statement of outcome |
| Reasoning | Why the decision was made |
| Evidence considered | What was reviewed |
| Grounds addressed | Response to each appeal ground |
| Further options | Any remaining rights |
| Implementation | What happens next |

### 5.3 Implementation

| Outcome | Implementation |
| :--- | :--- |
| Affirmed | No change; original decision enforced |
| Modified | New terms implemented within 5 days |
| Reversed | Refund/adjustment within 10 days |
| Remanded | Re-review within 15 days |

## 6. Appeal Levels

### 6.1 Standard Appeal Path

```
APPEAL ESCALATION PATH

Level 0: ORIGINAL DECISION
└─ By: Dispute Reviewer (Level 1)
└─ Appeal to: Level 1 Appeal

Level 1: FIRST APPEAL
└─ By: Senior Reviewer (Level 2)
└─ Appeal to: Level 2 Appeal (limited)

Level 2: FINAL APPEAL
└─ By: Dispute Manager (Level 3) or Trust Review Board
└─ Decision: FINAL
└─ No further appeal within platform

EXTERNAL: AFTER EXHAUSTING INTERNAL
└─ Arbitration clause may apply
└─ Or small claims court
└─ See Terms of Service
```

### 6.2 Appeal Authority

| Original Decision By | First Appeal To | Final Appeal To |
| :--- | :--- | :--- |
| Level 1 Reviewer | Level 2 Senior Reviewer | Level 3 Manager |
| Level 2 Senior Reviewer | Level 3 Manager | Trust Review Board |
| Level 3 Manager | Trust Review Board | Trust Review Board (panel) |
| Trust Review Board | Executive Appeal Panel | N/A (final) |

### 6.3 When Appeals Are Exhausted

| Stage | Status |
| :--- | :--- |
| After first appeal (standard) | May request second appeal |
| After second appeal | Internal appeals exhausted |
| Board decisions | Final unless executive escalation |
| Executive Panel | Absolutely final |

## 11. Educational Information

### 11.1 How to Write an Effective Appeal

```
APPEAL WRITING TIPS

DO:
• Be specific about what's wrong
• Reference the evidence
• Explain what outcome you want
• Stay factual and calm
• Include new evidence if available

DON'T:
• Make personal attacks
• Include unrelated issues
• Submit duplicate appeals
• Use threatening language
• Ignore deadlines
```

### 11.2 Common Mistakes to Avoid

| Mistake | Why It Hurts |
| :--- | :--- |
| Vague complaints | Reviewer can't address specifics |
| Missing deadline | Appeal may be denied |
| No new information | Same evidence, same result |
| Personal attacks | Distracts from merits |
