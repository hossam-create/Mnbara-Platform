## 4. Prohibited Practices

### 4.1 Staff Prohibitions

Platform staff, including support, **SHALL NOT**:

| Prohibition | Example of Violation |
| :--- | :--- |
| Recommend transactions | "You should buy now" |
| Advise on timing | "Wait until next week" |
| Predict outcomes | "You'll save money if..." |
| Suggest financial decisions | "That's a good/bad price" |
| Interpret financial data | "This means you should..." |
| Compare for decision | "Option A is better for you" |

### 4.2 Permitted Staff Actions

| Permitted | Example |
| :--- | :--- |
| Explain features | "Here's how escrow works" |
| Provide factual data | "The current price is $X" |
| Explain policies | "Our fee is 2.5% on all transactions" |
| Clarify information | "That rate is from Reuters" |
| Describe options | "You can choose escrow or card" |

### 4.3 Automated System Prohibitions

| System Feature | Prohibited |
| :--- | :--- |
| Price alerts | "Good time to buy" messages |
| Notifications | "Act now" urgency language |
| Comparisons | "Better deal than..." framing |
| Recommendations | Algorithmic "you should..." |
| Summaries | Inference-based conclusions |

## 6. Information vs. Advice Distinction

### 6.1 Information (Permitted)

| Type | Example | Why It's Information |
| :--- | :--- | :--- |
| Facts | "Current price: $1,199" | Objective observation |
| History | "Average last 30 days: $1,150" | Historical data |
| Options | "Payment methods: Escrow, Card" | Neutral listing |
| Features | "Escrow holds funds until delivery" | Explanation |
| Policies | "Fee is 2.5%" | Platform rule |

### 6.2 Advice (Prohibited)

| Type | Example | Why It's Advice |
| :--- | :--- | :--- |
| Recommendations | "You should use escrow" | Suggests action |
| Timing | "Wait for better prices" | Encourages timing |
| Comparison | "This is a good deal" | Evaluates for user |
| Predictions | "Prices will drop" | Future claim |
| Optimization | "Best value option is..." | Chooses for user |

## 7. When Users Ask for Advice

### 7.1 Redirecting Questions

When users ask for financial advice:

```
RESPONSE TEMPLATE

"Thank you for your question. 

We're not able to provide financial advice or recommendations 
on whether or what to buy. 

We can share factual information about:
• Current prices and availability
• How our features work
• Our fees and policies

For advice on financial decisions, we recommend consulting 
a qualified professional.

Is there factual information I can help you find?"
```

### 7.2 Common Questions & Responses

| User Asks | Correct Response |
| :--- | :--- |
| "Should I buy now?" | "We can't advise on timing. Current price is $X." |
| "Is this a good price?" | "We can't evaluate prices. 30-day average was $Y." |
| "Which traveler should I choose?" | "Here are the available travelers and their profiles." |
| "Will rates go up?" | "We can't predict rates. Current rate is Z." |
| "What's the best payment?" | "Options are escrow, card, etc. Each has different features." |
