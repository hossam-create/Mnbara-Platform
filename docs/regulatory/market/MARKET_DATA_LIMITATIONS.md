## 2. Data Types & Limitations

### 2.1 Price Data

| Attribute | Limitation |
| :--- | :--- |
| **Source** | User-submitted listings |
| **Real-time?** | No — reflects listing creation time |
| **Accuracy** | Not verified; user-submitted |
| **Completeness** | Platform listings only |
| **Market coverage** | Platform ecosystem only |

**Specific Limitations:**
- Prices are set by individual sellers/travelers
- Prices do not reflect broader market conditions
- Prices may not be updated if market changes
- We do not verify prices against external sources
- Listing prices may not equal final transaction prices

**Required Disclaimer:**
```
Price data reflects user-submitted listings only.
Prices are not verified against external market sources.
Actual transaction prices may differ.
```

---

### 2.2 Exchange Rate Data

| Attribute | Limitation |
| :--- | :--- |
| **Source** | Third-party provider (Reuters, XE, etc.) |
| **Real-time?** | Near real-time (up to 15-min delay) |
| **Accuracy** | Indicative only; not execution rate |
| **Coverage** | Major currency pairs |
| **Execution** | WE DO NOT EXECUTE FX |

**Specific Limitations:**
- Rates are indicative mid-market rates
- Actual rates are set by user's bank/payment provider
- Rates do not include bank spreads or fees
- Rates may be delayed by up to 15 minutes
- Weekend/holiday rates may be stale

**Required Disclaimer:**
```
Exchange rate is indicative only from [Source Name].
Your bank determines your actual rate.
Bank fees and spreads are not included.
Rate may be delayed up to 15 minutes.
```

---

### 2.3 Delivery Time Data

| Attribute | Limitation |
| :--- | :--- |
| **Source** | Historical platform data |
| **Accuracy** | Based on past deliveries |
| **Predictive?** | NO — historical only |
| **Factors** | Does not account for all variables |

**Specific Limitations:**
- Based on historical transactions only
- Does not predict your specific delivery
- Does not account for: customs delays, weather, carrier changes
- Averages may not reflect recent changes
- Individual results vary significantly

**Required Disclaimer:**
```
Delivery times are historical averages only.
Your actual delivery time may differ significantly.
Customs, weather, and other factors affect delivery.
```

---

### 2.4 Availability Data

| Attribute | Limitation |
| :--- | :--- |
| **Source** | Real-time platform data |
| **Accuracy** | Current at time of query |
| **Volatility** | Changes frequently |
| **Commitment** | Availability not guaranteed |

**Specific Limitations:**
- Travelers may become unavailable at any time
- Displayed availability is not a commitment
- Matching is not guaranteed
- Availability changes without notice

**Required Disclaimer:**
```
Availability shown is current at time of viewing.
Travelers may become unavailable at any time.
Availability does not guarantee matching.
```

---

### 2.5 Trust/Reputation Data

| Attribute | Limitation |
| :--- | :--- |
| **Source** | Platform activity data |
| **Accuracy** | Reflects tracked activity only |
| **Completeness** | Platform history only |
| **Predictive?** | NO — historical only |

**Specific Limitations:**
- Trust scores reflect platform history only
- Does not capture off-platform behavior
- Scores are not guarantees of future performance
- New users have limited history
- Scores may change without notice

**Required Disclaimer:**
```
Trust scores reflect platform history only.
Scores are not guarantees of future performance.
You should review profiles and exercise judgment.
```

## 3. Data Source Transparency

### 3.1 Source Disclosure Requirements

Every data element must disclose:

| Element | Requirement |
| :--- | :--- |
| Source name | Named or described |
| Update frequency | How often data refreshes |
| Delay | If applicable |
| Limitations | Material limitations |

### 3.2 Source Categories

| Category | Disclosure |
| :--- | :--- |
| **Platform-generated** | "Based on platform data" |
| **Third-party licensed** | Name the provider |
| **User-submitted** | "User-submitted, unverified" |
| **Calculated/derived** | Explain calculation |

## 4. Data Quality Warnings

### 4.1 Staleness Warnings

| Data Age | Warning |
| :--- | :--- |
| < 5 minutes | None required |
| 5-60 minutes | "Data may not be current" |
| 1-24 hours | "Data from [time] ago" |
| > 24 hours | Do not display OR "Significantly outdated" |

### 4.2 Low Data Warnings

| Condition | Warning |
| :--- | :--- |
| < 5 data points | "Limited data available" |
| New route/category | "Insufficient history" |
| Data gap | "Incomplete coverage" |

### 4.3 High Variance Warnings

| Condition | Warning |
| :--- | :--- |
| High price variance | "Prices vary significantly" |
| Inconsistent delivery times | "Delivery times vary widely" |
| Fluctuating rates | "Rate has changed significantly recently" |

• Delivery times for any transaction

fees paid by user in the preceding 12 months.
