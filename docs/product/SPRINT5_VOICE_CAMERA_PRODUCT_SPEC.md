### 2.3 Accessibility Support

**Use Case:** Users with visual or motor impairments navigate via voice.

**Supported Commands:**
| Command | Action |
| :--- | :--- |
| "Go to my orders" | Navigate to orders screen |
| "Read this page" | Screen reader activation |
| "Search for [item]" | Open search with query |
| "Go back" | Previous screen |
| "Help" | Open assistance menu |

**Rules:**
- Voice navigation is opt-in
- All actions have audio confirmation
- No financial actions via voice only (confirmation required on-screen)
