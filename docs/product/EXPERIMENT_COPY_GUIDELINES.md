## 3. Forbidden Copy Patterns

### 3.1 Urgency Language

| Forbidden | Why | Alternative |
| :--- | :--- | :--- |
| "Only 2 left!" | Creates artificial scarcity | "2 travelers available on this route" |
| "Act fast!" | Pressures decision | "Take your time to decide" |
| "Limited time!" | Artificial urgency | [Remove entirely] |
| "Don't miss out!" | FOMO manipulation | [Remove entirely] |
| "Hurry!" | Pressure tactic | [Remove entirely] |

### 3.2 Social Proof Manipulation

| Forbidden | Why | Alternative |
| :--- | :--- | :--- |
| "Most people choose X" | Influences choice | "Here are your options" |
| "Recommended by 95% of users" | Statistical manipulation | [Remove entirely] |
| "The smart choice" | Implies other choices are dumb | "One option is..." |
| "Trusted by thousands" | Vague social proof | Specific: "47 completed transactions" |

### 3.3 Fear-Based Language

| Forbidden | Why | Alternative |
| :--- | :--- | :--- |
| "Protect yourself!" | Fear tactic | "Your payment is protected by escrow" |
| "Don't risk it!" | Fear tactic | "Here's how protection works" |
| "Danger!" | Excessive alarm | "Something to know" |
| "You could lose everything" | Exaggeration | "Limited dispute options available" |

### 3.4 Manipulative Comparisons

| Forbidden | Why | Alternative |
| :--- | :--- | :--- |
| Bigger button for preferred option | Visual manipulation | Equal-sized buttons |
| Greyed-out "dismiss" | Makes dismissal hard | Equal prominence |
| Hiding the "no thanks" option | Force acceptance | Clear, equal alternatives |
| "Basic" vs "Premium" labels | Implies hierarchy | Neutral labels |

## 5. Copy Review Checklist

Before publishing any experiment copy:

### Neutrality Check
- [ ] Does NOT recommend one option over another
- [ ] Does NOT use superlatives ("best", "safest", "smartest")
- [ ] Does NOT imply a wrong choice exists
- [ ] Equal prominence for all options

### Honesty Check
- [ ] All statements are factually correct
- [ ] No exaggeration or minimization
- [ ] Limitations are disclosed
- [ ] Source of data is clear (if applicable)

### Clarity Check
- [ ] Sentences are under 20 words
- [ ] No jargon or technical terms (or they're explained)
- [ ] Action is clear
- [ ] User knows what happens next

### Pressure Check
- [ ] No urgency language
- [ ] No scarcity tactics
- [ ] No social proof manipulation
- [ ] No fear-based messaging

### Accessibility Check
- [ ] Reading level: Grade 8 or below
- [ ] No idioms that don't translate
- [ ] No culturally specific references
- [ ] Clear even out of context

## 6. Word Choice Reference

### Preferred Words

| Instead of | Use |
| :--- | :--- |
| Buy now | Proceed / Continue |
| Recommended | One option is |
| Best | [Don't rank] |
| Protect yourself | Your payment is protected |
| Don't risk | Here's something to know |
| Limited | [Remove or be specific] |
| Exclusive | Available |
| Premium | [Use descriptive term] |
| Free | No additional cost |
| Guarantee | [Explain the protection specifically] |

### Neutral Action Labels

| Situation | Neutral CTA |
| :--- | :--- |
| Choose payment | "Continue with [method]" |
| Select traveler | "Request this traveler" |
| Complete purchase | "Confirm and pay" |
| Read more | "Learn more" / "See details" |
| Dismiss modal | "Close" / "Not now" |
| Skip optional | "Skip for now" |

## 7. Tone Calibration

### Education Screens
**Tone:** Friendly teacher

```
"Let's walk through how this works..."
"Here's what happens when you..."
"This means that..."
```

### Decision Screens
**Tone:** Neutral facilitator

```
"Here are your options..."
"You can choose..."
"Both options are available..."
```

### Warning Screens
**Tone:** Calm advisor

```
"Here's something to know..."
"Before you proceed..."
"You might want to consider..."
```

### Confirmation Screens
**Tone:** Clear summarizer

```
"You're about to..."
"Here's what will happen..."
"Once you confirm..."
```

## 8. Cultural Considerations

### Language Variations (for localization)
- Avoid idioms that don't translate literally
- Avoid humor (translates poorly)
- Avoid sports/cultural metaphors
- Be explicit about currency and dates

### Regional Sensitivity
- Avoid assumptions about payment access (not everyone has cards)
- Avoid assumptions about banking hours (time zones)
- Respect different trust norms (face-to-face cultures vs. online-first)
