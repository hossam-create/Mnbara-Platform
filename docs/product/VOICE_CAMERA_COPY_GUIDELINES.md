```

```

```

- 'Search for' and an item name
- 'Go to orders'
- 'Go back'
```

## 7. Word Choice Reference

### Preferred Language

| Instead of | Use |
| :--- | :--- |
| "We're listening" | "Listening..." |
| "Talk to me" | "Speak your request" |
| "I heard you say" | "You said" |
| "I think this is" | "This might be" / "We suggest" |
| "We know this is" | "We think this could be" |
| "Trust our suggestion" | "Please verify this is correct" |
| "Perfect!" | "Got it" / "Understood" |
| "Smart camera" | "Photo help" |
| "AI-powered" | [Don't mention AI] |

### Forbidden Words

| Never Use | Why |
| :--- | :--- |
| "Always listening" | Implies surveillance |
| "Watching" | Creepy |
| "Detected you" | Invasive |
| "Identified" (for people) | Face recognition implication |
| "Tracking" | Surveillance language |
| "Automatically" | Implies no control |
| "Smart" (for mic/camera) | Overpromises |

## 9. Copy Review Checklist

Before publishing voice/camera copy:

### Consent Language
- [ ] Explains what we capture
- [ ] Explains when we capture
- [ ] Explains what we do with it
- [ ] Explains what we DON'T do
- [ ] "Not now" is equally prominent as "Allow"

### Suggestion Language
- [ ] Clearly labeled as suggestion
- [ ] Includes uncertainty ("might be", "could be")
- [ ] Asks user to verify
- [ ] Edit option is prominent

### Privacy Language
- [ ] Explains data deletion
- [ ] No surveillance words
- [ ] User control emphasized
- [ ] Delete option clearly available

### Accessibility
- [ ] All buttons have labels
- [ ] Recording state is announced
- [ ] Error messages are screen reader friendly
